 ##Стартиране на проекта
 <ul>
 	<li>
 		1. Стартирайте локална база : mongod --dbpath {Project_PATH}/data 
 	</li>
	<li>
 		2. Инсталирайте node_modules 
			cd {Project_PATH};
			npm install -save
 	</li>
	<li>
 		2. Пуснете приложението
			cd {Project_PATH};
			node server
 	</li>
 </ul>
